# BAS1C VISUALS Portfolio

React/Vite portfolio site with an Express server for production hosting.

## Local setup

1. Install dependencies:
   `npm install`
2. Start development:
   `npm run dev`

## Build

`npm run build`

## Start production bundle

`npm start`

The server uses `process.env.PORT` when provided by the host, otherwise it falls back to `3000`.

The contact form opens WhatsApp with structured project details. No backend email setup is required.
